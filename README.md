# Comparador de Precios (Android Nativo)

Aplicación nativa de Android para comparar precios de supermercados, construida con **Jetpack Compose**, **Room**, **MVVM** y **ML Kit (Barcode Scanning)**.

## Características
- 100% Funcional Offline.
- Gestión de Supermercados con direcciones personalizadas.
- Gestión de Productos con soporte para distintos tipos de unidades (Pieza, Kg, Litro, Empaque).
- Escaneo de códigos de barras/QR para búsqueda y registro rápido.
- Comparación automática resaltando el precio unitario más bajo.
- Sistema de respaldo (Exportar/Importar base de datos).
- Interfaz moderna inspirada en "Nothing Phone".

## Requisitos
- Android Studio Ladybug o superior.
- SDK de Android 36 (o compatible).
- Kotlin 2.2.10.

## Instalación
1. Clona este repositorio.
2. Abre el proyecto en Android Studio.
3. Sincroniza Gradle.
4. Ejecuta en un dispositivo físico o emulador.
